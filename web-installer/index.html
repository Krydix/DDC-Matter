Install Latest Firmware

Connect the ESP32 over USB, close any serial monitor, then start the installer.

Use desktop Chrome or Edge over HTTPS to flash over Web Serial. This page must be opened over HTTPS or localhost for Web Serial to work.

After flashing, use the device’s BLE commissioning flow, then open the web UI exposed by the firmware.

Before You Start

  1. Use a USB data cable, not a charge-only cable.
  2. Disconnect any app using the serial port, including terminal monitors.
  3. If multiple serial devices are attached, choose the ESP32 when prompted.
  4. After install, replug the board if your browser does not reconnect automatically.
